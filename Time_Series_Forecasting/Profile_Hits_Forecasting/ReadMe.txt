This folder contains the python code for 
Forecasting Profile Hits creating a dummy dataset from 2014
profile_hits.py: Contains code for forecasting the profile hits for a particular user so the user can see the predicting based on the past hits and showing it in graphical format. it uses the ARIMA model for doing the prediction model.

No train test split is used since all the data is dummy(not realtime) it doesnt make sense to check it.


The Code is written in python using Anaconda Navigators Sypder.
